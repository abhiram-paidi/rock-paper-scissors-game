# 🎮 Rock Paper Scissors Game

A modern and responsive Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## 🚀 Features

- ✊ Rock, ✋ Paper, ✌️ Scissors gameplay
- 📱 Fully Responsive Design
  - Desktop
  - Tablet
  - Mobile
- 🎬 Hand Collision Animation
- ⏳ Rock → Paper → Scissors Countdown Effect
- 🏆 Win / Lose / Draw Detection
- 📊 Live Score Tracking
- 💾 Local Storage Support
- ⌨️ Keyboard Shortcuts (Desktop)
  - R = Rock
  - P = Paper
  - S = Scissors
- 🔒 Buttons Disabled During Animation
- 🎨 Color-Coded Results
  - Green → Win
  - Red → Lose
  - Yellow → Draw
- 🔄 Reset Game Functionality
- 🖥️ Structured Layout
  - Header
  - Main Body
  - Footer

## 🎮 How to Play

### Mouse Controls

1. Click Rock, Paper, or Scissors.
2. Watch the countdown animation.
3. The computer randomly selects a move.
4. The result is displayed.
5. Scores are automatically updated.

### Keyboard Controls (Desktop)

```text
R → Rock
P → Paper
S → Scissors
```

---

## 💾 Local Storage

Scores are automatically saved in the browser.

Even after refreshing the page:

- Wins
- Losses
- Draws

remain stored until the Reset Game button is pressed.

### Desktop

### Mobile

## 🔄 Reset Game

The Reset Game button:

- Resets all scores
- Clears Local Storage
- Resets both hands to ✊
- Resets the result message
